# AI Puzzle Solver

AI Puzzle Solver is a Streamlit-based project that solves multiple puzzle types using classical Artificial Intelligence search algorithms.

## Project Overview

This project creates a puzzle-solver that efficiently tackles diverse puzzles using multiple search algorithms, with customized heuristics for informed searches. It offers user-friendly input, algorithm selection, and performance comparison, with thorough documentation for analysis.

## Supported Puzzles

- 8 Puzzle
- Maze
- Sudoku

## Algorithms

- Breadth First Search
- Depth First Search
- Depth Limited Search
- Iterative Deepening DFS
- Uniform Cost Search
- Greedy Best First Search
- A* Search
- Beam Search

## Heuristics

- 8 Puzzle: Manhattan Distance
- Maze: Manhattan Distance
- Sudoku: Empty Cell Count

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
